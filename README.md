# 🥗 Fast Diet e Treinos - Landing Page

Esta é a página oficial de vendas do projeto **Fast Diet e Treinos**. O objetivo deste site é centralizar a venda de produtos de emagrecimento e a oferta de consultoria fitness de forma profissional, rápida e responsiva.

## 🚀 Funcionalidades

- **Venda de Mentorias:** Botão integrado diretamente com o checkout da **Kiwify**.
- **Venda de Produtos:** Botão "Converse com o Vendedor" vinculado ao **WhatsApp** com mensagem pré-definida.
- **Prova Social:** Seção de depoimentos com vídeos incorporados diretamente do **Google Drive**.
- **Design Responsivo:** O site adapta-se automaticamente a telemóveis, tablets e computadores.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação de conteúdo.
- **CSS3:** Estilização, layout (Flexbox) e responsividade (Media Queries).
- **Google Drive API (Embed):** Hospedagem de vídeos leve e eficiente.
